Supabase 연결 설정
처음 한 번만 설정하면 이 기기에 저장됩니다. Supabase 계정이 없으면 supabase.com에서 무료로 만드세요.
1. supabase.com → 새 프로젝트 생성
2. SQL Editor → supabase_schema.sql 실행
3. Settings → API Keys → Publishable key 복사
⚠ Publishable key는 RLS(Row Level Security)가 켜진 경우 안전합니다. schema.sql 실행 후 연결하세요.
